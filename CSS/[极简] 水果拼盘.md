>

>

## 题解

```css
/* 菠萝 TODO 待补充代码 */
.yellow {
	/* 改变菠萝的排序到最后 */
	order: 1;

	/* 改变菠萝的垂直排序, 为底部对齐 */
	align-self: flex-end;
}
```
